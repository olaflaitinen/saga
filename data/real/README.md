# data/real/ -- documentation only

This directory contains no data files.

The real LISA microdata are restricted to approved researchers operating inside the
Statistics Sweden MONA secure compute environment under project SCB-MONA-2026-147.
No row-level data or intermediate outputs linked to identifiable individuals may be stored
outside the MONA environment.

See [docs/data/mona-secure-environment.md](../../docs/data/mona-secure-environment.md)
for the access procedure.
