# Dual-use statement

## Intended use

SAGA is released as a research artifact for academic reproducibility and scientific
dissemination. Its intended use is probabilistic earnings forecasting and microsimulation
research in academic and public-policy contexts.

## Explicitly prohibited uses

The authors of SAGA explicitly prohibit the following uses of the SAGA model weights,
synthetic mirror dataset, and source code:

1. **Credit scoring or credit decisioning.** Using SAGA's earnings forecasts to inform
   lending, mortgage, or consumer credit decisions for individual applicants.

2. **Insurance pricing.** Using SAGA's lifetime earnings distributions to price life, income
   protection, or annuity products for individual policyholders.

3. **Employment screening.** Using SAGA's earnings forecasts as a factor in hiring, promotion,
   compensation, or termination decisions for individual employees or job applicants.

4. **Tax enforcement targeting.** Using SAGA's earnings predictions to select individuals
   for tax audits or to set enforcement priorities.

5. **Social benefit rationing.** Using SAGA's lifetime earnings forecasts to determine
   eligibility for or the amount of any social benefit, pension, or insurance payment for
   individual claimants.

## License-level enforcement

The Apache 2.0 license under which the SAGA code is released does not prohibit commercial
use. However, this dual-use statement constitutes a public ethical commitment by the authors,
and the authors request that all users of the SAGA codebase, model weights, and synthetic
mirror dataset respect these restrictions. The authors will not provide technical support for
uses that violate this statement.

The synthetic mirror dataset and model documentation are released under Creative Commons
Attribution-NonCommercial 4.0 International (CC BY-NC 4.0), which prohibits commercial use
of those materials. Any commercial application of the synthetic mirror or documentation
requires separate written permission from the authors.
